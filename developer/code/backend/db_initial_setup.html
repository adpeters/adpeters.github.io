db_initial_setup¶

flask_cc.db_initial_setup.db_initial_setup(admin_user_password)[source]¶

Loads objects to be created in the database on initial creation.

Notes: - Objects are specified in json file db_initial_setup.json. - To specify an object to be loaded in development, test, or production only

use an “environments” property. It can be a string or a list. Possible values are: * development * test * production * all Default value is all if not specified.
flask_cc.db_initial_setup.fill_vars(value)[source]¶

Fills a variable specified in the json with a value coming from either the key store or environment variables.

Notes: Tries to fill from key store first, then environment variables.

Parameters:value (str) – Variable name to be found in key store or environment variables.
Return type:str
Returns:The value found for the given variable.
