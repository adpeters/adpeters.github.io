azure_kv¶

flask_cc.azure_kv.get_secret(secret_name, default_value=None)[source]¶

Get a secret from the Azure Key Vault.

Parameters:
  • secret_name (str) – Name of the secret to get the value for from KeyVault.
  • default_value (str) – Value to return if secret is not found. If None, will raise ResourceNotFoundError if value not found.
Return type:

str
Returns:

Value of secret if found or default_value if not.
