flask_cc.api.views.get_report_url(specs_data, in_specs, in_config, in_jrxml, output, extra_files=None)[source]¶

Generates a PDF report using Mapfish and return link to it.

Parameters:
  • specs_data (dict) – Dictionary of data to be written to the specs file.
  • in_specs (str) – Name of specs file.
  • in_config (str) – Name of config file.
  • in_jrxml (str) – Name of template file.
  • output (str) – Name of output file.
  • extra_files (dict) – Dictionary of file_name: data pairs to be created for this report.
Return type:

str
Returns:

URL to PDF report.
